# Pop-in-Solutions Website Redesign

## Project Overview

This project is a responsive multi-page website redesign for Pop-in-Solutions, a DevSecOps-driven IT solutions provider based in South Africa.

The website was developed using HTML, CSS, and JavaScript and includes:

* Responsive design
* Dark/light mode
* Scroll reveal animations
* Mobile hamburger navigation
* Multi-page layout

---

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Visual Studio Code

---

## Website Pages

* Home
* About
* Services
* Contact

---

## Features

* Responsive mobile navigation
* Animated scroll reveal effects
* Theme toggle (dark/light mode)
* Contact form UI
* Modern glassmorphism design

---

## How to Run the Website

1. Download the project files
2. Open the project folder
3. Launch `index.html` in a web browser
